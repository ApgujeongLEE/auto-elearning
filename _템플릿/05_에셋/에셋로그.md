# 에셋 로그

---

## 형식

```
### {파일 경로}
- 종류: 이미지 / 일러스트 / BGM / SFX / 폰트 / B-roll
- 출처: (URL 또는 구입처)
- 작가/제공자:
- 라이선스: (예: CC BY 4.0, 유료 구입, 자체 제작)
- 사용 가능 범위: (상업적/비상업적, 크레딧 필요 여부)
- 추가일: YYYY-MM-DD
- 비고:
```
