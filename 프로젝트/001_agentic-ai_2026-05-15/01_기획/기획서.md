## 1. 학습 목표 (Learning Objectives)

시청 완료 시 학습자가 할 수 있어야 하는 것.

- [ ] **에이전틱 AI**가 기존 챗봇 AI와 무엇이 다른지 한 문장으로 설명할 수 있다.
- [ ] 에이전틱 AI의 **4가지 핵심 능력**(계획·도구 사용·기억·자기 점검)을 떠올릴 수 있다.
- [ ] 에이전틱 AI가 **현재 어디까지 와 있는지**(시장 트렌드·실제 사례) 감을 잡는다.

## 2. 타겟 학습자 (Target Audience)

- **사전 지식**: ChatGPT 같은 챗봇 AI를 한 번이라도 써본 일반인. 기술 배경 불필요.
- **연령/직군**: 20~50대 직장인·자영업자·학생. 비전공자 포함.
- **학습 동기**: "요즘 자꾸 들리는 '에이전트 AI'가 대체 뭐길래?" 라는 호기심.
- **불편 지점 (Pain Point)**: 용어가 추상적이고, 챗봇과 뭐가 다른지 직관적으로 잘 안 와닿음.

## 3. 분량 및 구조 (Length & Structure)

- **목표 분량**: 1분 (60초 ±5초)
- **구성**:
  - 인트로/후킹 0:00~0:10 (10초)
  - 정의 (기존 AI 대비 차이) 0:10~0:22 (12초)
  - 4가지 핵심 능력 0:22~0:34 (12초)
  - 시장 통계 (Gartner) 0:34~0:46 (12초)
  - 실제 사례 0:46~0:56 (10초)
  - CTA 0:56~1:00 (4초)
- **씬 개수**: 6개 (S01 ~ S06)
- **이유**: 1분 영상은 씬당 평균 8~12초가 시청자 집중에 가장 적합. 6씬이면 정보 밀도와 호흡이 균형.

## 4. 핵심 키워드 (Keywords)

- **1순위**: `에이전틱 AI`, `Agentic AI`, `AI 에이전트`
- **2순위**: `자율 AI`, `AI 자동화`, `LLM 에이전트`
- **3순위**: `Gartner 2025`, `Claude Code`, `생성형 AI`

## 5. 톤매너 (Tone & Manner)

- **말투**: 친근한 존댓말 (해요체). "이거 알면 좋아요" 톤. 친구가 신기한 거 설명해 주는 느낌.
- **속도**: 분당 **300~340자** (평소보다 약간 빠르게 — 1분 안에 정보 밀도 확보 위해)
- **분위기**: 호기심 자극 + 약간의 활기. 인트로는 약간 빠르고, 통계 구간은 또박또박.
- **금지 표현**: "혁명적", "완벽한", "쉽죠?", "당연히" — 과장 또는 학습자 좌절 유발.

## 6. 비주얼 방향 (Visual Direction)

- **얼굴 노출**: 없음 (전 영상 모션 그래픽으로 구성).
- **메인 비주얼 4종 (사용자 요청 반영)**:
  1. **비교 도식** (S02) — 좌: "Q → A" 단순 화살표 / 우: "Goal → Plan → Tool → Tool → Result" 다단계 파이프라인. 좌우 분할 화면 + 모션으로 흐름 진행.
  2. **4 능력 아이콘 그리드** (S03) — 2×2 그리드. 계획(체크리스트 아이콘) / 도구(렌치 아이콘) / 기억(뇌+저장 아이콘) / 자기 점검(돋보기 아이콘). 발화 순간마다 하나씩 컬러로 채워짐.
  3. **통계 콜아웃 + 차트** (S04) — "#1 트렌드" 거대 숫자 + "Gartner Strategic Tech Trends 2025" 캡션. 옆에 시장 규모/도입률 막대 차트가 차오르는 모션.
  4. **사례 로고 그리드** (S05) — Claude Code, OpenAI Operator, Microsoft Copilot, Google Gemini Agent 등 잘 알려진 에이전틱 AI 제품들의 텍스트/심플 마크 그리드 (로고 직접 사용은 라이선스 확인 필요 — 텍스트 표기 권장).
- **B-roll**: 키보드 타이핑, 알림 팝업 등 짧은 5~10프레임 컷 (1~2회만, 강조용)
- **컬러**: `공용자산/브랜드/컬러팔레트.md` 참조. 메인 강조는 Brand Blue, 통계 콜아웃 숫자는 Brand Yellow.
- **폰트**: Pretendard (본문/타이틀), JetBrains Mono (코드 톤 텍스트 — "Goal → Plan → Tool" 같은 파이프라인)
- **자막**: 항상 ON, 하단 중앙, 검정 반투명 박스 + 흰 글씨. 1분 영상이라 자막 가독성 매우 중요.
