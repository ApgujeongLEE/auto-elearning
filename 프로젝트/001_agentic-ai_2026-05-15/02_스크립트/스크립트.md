## 발성 가이드 범례

- `**볼드**` : 강조 (목소리 톤 올림)
- `_이탤릭_` : 부드럽게, 톤 다운
- `//` : 짧은 포즈 (0.3초 호흡)
- `///` : 긴 포즈 (0.8초 호흡, 화면 전환 신호)
- `[들숨]`, `[웃음]` : 비언어 지시

**전체 톤**: 1분 영상이라 평소보다 약간 빠르게 (분당 300~340자). 호흡 표시(`//`, `///`)에서만 확실히 쉬고, 나머지는 끊지 말고 흐름 유지.

## S02 — 정의: 기존 AI vs 에이전틱 AI

**[메타]**
- 추정 타임코드: 00:10 ~ 00:22 (12초)
- 발성 톤: 차분, 또박또박 (개념 도입이라 명확하게)
- 핵심 키워드: `에이전틱 AI`, `목표`, `여러 단계`
- 시각 강조: 좌우 분할 화면 — 좌측 "Q→A" 단순 화살표, 우측 "Goal → Plan → Tool → Tool → Result" 파이프라인이 흐르며 차오름

### 📢 읽을 대사 (S02)
```text
이게 바로 // **에이전틱 AI**예요. ///
기존 챗봇이 // 질문 하나에 답 하나라면, //
에이전틱 AI는 // **목표 하나**에 //
_스스로_ 여러 단계를 실행해요. ///
```

- 시각 강조: 2×2 아이콘 그리드 — 발화 순간마다 하나씩 컬러로 채워짐 (체크리스트/렌치/뇌+저장/돋보기)

### 📢 읽을 대사 (S03)
```text

**기억**을 유지하면서, //

```
