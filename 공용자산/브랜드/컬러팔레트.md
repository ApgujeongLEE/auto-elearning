# 브랜드 컬러 팔레트

> 실제 브랜드 컬러가 정해지면 아래 값을 교체하세요. 현재는 안전한 기본값입니다.

---

## 메인 컬러 (1개, 강조 키워드용)

| 이름 | HEX | RGB | 용도 |
|------|-----|-----|------|
| Brand Blue | `#2563EB` | 37, 99, 235 | 핵심 키워드 강조, 타이틀 액센트 |

## 서브 컬러 (1개, 대비/2차 강조)

| 이름 | HEX | RGB | 용도 |
|------|-----|-----|------|
| Brand Yellow | `#FACC15` | 250, 204, 21 | 서브 키워드, 하이라이트 |

## 텍스트 컬러

| 이름 | HEX | 용도 |
|------|-----|------|
| Text Primary | `#1F2937` | 본문 (밝은 배경) |
| Text Secondary | `#6B7280` | 보조 텍스트, 캡션 |
| Text Inverted | `#FFFFFF` | 어두운 배경/자막바 |

## 배경 컬러

| 이름 | HEX | 용도 |
|------|-----|------|
| BG Default | `#FFFFFF` | 기본 흰 배경 |
| BG Soft | `#F8F9FA` | 카드/섹션 분리 |
| BG Dark | `#111827` | 코드 화면, 어두운 인트로 |

## 시스템 컬러 (제한적 사용)

| 이름 | HEX | 용도 |
|------|-----|------|
| Success | `#10B981` | 정답/성공 표시 |
| Warning | `#F59E0B` | 주의 |
| Error | `#EF4444` | 오류/금지 (자주 쓰지 말 것) |

## 자막바

- 배경: `rgba(0, 0, 0, 0.6)` (검정 60% 투명)
- 글자: `#FFFFFF`

## 사용 원칙

- 한 화면에 동시 강조 컬러는 **최대 2개** (메인 + 서브)
- 빨강 (Error)은 진짜 경고에만, 일반 강조에 쓰지 말 것
- 다크 모드 배경 사용 시 텍스트는 Text Inverted (`#FFFFFF`) 사용
- 그라데이션은 메인 → 서브 (블루 → 옐로우) 한 방향만 사용
