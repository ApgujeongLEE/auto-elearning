# 브랜드 가이드라인

브랜드 정체성을 영상에 일관되게 반영하기 위한 규칙. 컬러/폰트/로고의 구체 값은 별도 문서 참조.

---

## 1. 로고 사용

- 로고 파일: `로고/logo_light.svg`, `로고/logo_dark.svg`
- 영상 우상단 워터마크: 가로 약 160px, opacity 30%
- 인트로/아웃트로 중앙 배치 시: 가로 480px, opacity 100%
- 변형 금지: 색상 임의 변경, 회전, 왜곡, 그림자 추가 금지

## 2. 보이스 톤

- 친근한 존댓말 (해요체)
- 학습자를 동등한 위치에서 안내하는 톤 (선생-학생 구도 X)
- "여러분", "우리" 호칭 사용 가능. "당신"은 사용 안 함
- 전문 용어는 처음 등장 시 한 번 풀어 설명

## 3. 시각 정체성

- **여백을 충분히** — 한 화면에 정보 과밀 금지
- **타이포 중심** — 화려한 일러스트보다 명확한 텍스트 우선
- **차분한 모션** — 튕김/회전은 강조 1~2회만, 본문은 페이드/슬라이드
- **자막은 영상의 일부** — 후처리가 아니라 디자인 요소로 인식

## 4. 콘텐츠 톤

- 학습 가치 명시 ("이 영상 보면 무엇을 할 수 있게 되는지" 인트로에)
- 실패담/시행착오 적극 활용 (학습자 공감)
- 과장 금지 ("혁명적", "최고의", "완벽한" 등)
- 단정 표현 자제 ("쉽죠?", "당연히" 금지)

## 5. 변경 이력

- 가이드라인 변경 시 `CHANGELOG.md` 에 기록
- 변경 사항은 진행 중인 프로젝트에 소급 적용하지 않음 (시각 일관성 깨짐 방지)
