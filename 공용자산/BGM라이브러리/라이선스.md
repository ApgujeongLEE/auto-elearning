# BGM 라이선스 기록

이 라이브러리에 추가하는 모든 트랙의 출처와 라이선스를 반드시 기록한다.

---

## 형식

```
### {파일명}
- 카테고리: calm / upbeat / tension
- 출처: (URL 또는 구입처)
- 작곡가/아티스트:
- 라이선스: (예: Creative Commons BY 4.0, 구입 라이선스, 자체 제작)
- 사용 가능 범위: (예: YouTube/상업적 사용 가능, 크레딧 표기 필요 여부)
- 추가일: YYYY-MM-DD
- 비고:
```

---

## 등록된 트랙

(트랙이 추가되면 여기에 기록)

---

## 사용 시 주의

- 라이선스가 명시되지 않은 트랙은 영상에 사용 금지
- 크레딧 표기가 필요한 경우 영상 설명란에 명시
- 상업적 사용 불가 트랙은 별도 표시 후 학습용 데모에만 사용
